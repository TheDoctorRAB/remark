# 2.3 Statistical moments [Expected value and Variance](http://www.statlect.com/ucdexp1.htm) The PDF for the exponential distribution is given as: $\rho (x) \equiv \lambda e^{-\lambda x}, x > 0$ Then the expected value is derived as: $E[X] \equiv \int_{0}^{\infty} x \rho (x) dx = \frac{1}{\lambda}$ And the variance is derived as: $Var[X] \equiv \int_{0}^{\infty} x^2 \rho (x) dx = \frac{1}{\lambda^2}$
