[50] An inspector selects 50 containers of uranium oxide powder at random to weigh. The goal is to compare the weight from the equipment the inspector is using to the book weight provided by the operator. If there is a difference greater than 50 g, this is labeled a defect. Within the context of safeguards, this might indicate that the operator is cooking the books to divert some material. The inspector finds 3 defects. Let $p$ be defined as the fraction of defects in the selected population. Assume the probability density function for the number of defects is a binomial distribution. Test the hypothesis that $p \leq 0.045$ against $p > 0.045$ with $\alpha = 0.05$. How many containers should be reweighed to detect $p = 0.12$ with a probability of 0.85 or greater? In the original 50 sample size, how many defects can exist before rejecting the hypothesis of $p = 0$? What is the value of $\alpha$? If there are no defects found, can you conclude $p = 0$?
